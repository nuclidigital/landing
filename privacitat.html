Política de privacitat i galetes

1. Responsable del tractament

Roger Gibaja, titular del lloc personal nuclidigital.com. Contacte per a qüestions de privacitat: hola@nuclidigital.com.

2. Quines dades es tracten

Aquest Web no recull dades personals per defecte: no hi ha formularis, comptes ni rastreig propi. Els únics tractaments possibles són:

  • Correu electrònic: si escrius a l'adreça de contacte, es tractaran les dades que incloguis al missatge amb l'única finalitat d'atendre'l (base legal: consentiment / interès legítim en respondre). Es conserven el temps imprescindible.
  • Registres tècnics de l'allotjament: el Web s'allotja a GitHub Pages; GitHub pot registrar adreces IP amb finalitats de seguretat segons la seva política de privacitat. El Titular no hi té accés.
  • Analítica amb consentiment: vegeu l'apartat 4.

3. Emmagatzematge local i galetes

El Web utilitza únicament emmagatzematge local tècnic del navegador per recordar la teva elecció de privacitat. Només si acceptes la categoria d'analítica s'instal·len galetes de Google Analytics:

4. Analítica (Google Analytics 4)

Amb el teu consentiment previ, el Web pot utilitzar Google Analytics 4 (Google Ireland Ltd.) per obtenir estadístiques d'ús agregades. S'aplica Google Consent Mode v2 amb tots els senyals denegats per defecte, anonimització d'IP i cap finalitat publicitària. El Web respecta els senyals Global Privacy Control i Do Not Track del navegador. Les transferències internacionals es regeixen pel Data Privacy Framework UE–EUA i clàusules contractuals tipus de Google.

5. Drets de les persones

Pots exercir els drets d'accés, rectificació, supressió, oposició, limitació i portabilitat escrivint a hola@nuclidigital.com. També tens dret a presentar una reclamació davant l'Agència Espanyola de Protecció de Dades o l'Autoritat Catalana de Protecció de Dades.

6. Canvis en aquesta política

Qualsevol canvi substancial s'anunciarà en aquesta pàgina actualitzant la data i la versió, i, si escau, es tornarà a demanar el consentiment.
